# The Mortgage Mailbox Project

An interactive timeline documenting 49 unsolicited mortgage solicitations received by a single veteran borrower following a VA purchase loan closing on January 7, 2025 with Rocket Mortgage in Summit County, Ohio.

All third-party companies shown have no affiliation with the VA, Rocket Mortgage, or the U.S. government. Borrower information was sourced entirely from public county deed and mortgage recording records.

**Live page:** https://chris-eidson.github.io/mortgage-mailbox-project/

Built with vanilla HTML, CSS, and JavaScript. No frameworks or external dependencies.
